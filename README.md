# 📘 Introdução à Computação Estruturada em C

## 📖 Sobre o repositório

Este repositório tem como finalidade o registro das atividades e exercícios de código desenvolvidos na disciplina de **Introdução à Computação Estruturada em C**.

## 🎯 Objetivo

Organizar e versionar todo o progresso ao longo da disciplina, facilitando a revisão dos conteúdos e o acompanhamento da evolução no aprendizado da linguagem C.

## 🧠 Conteúdo abordado

Neste repositório você encontrará:

* 📝 Questões de exercícios
* 💡 Desafios de código
* 📚 Tópicos e conceitos relacionados à disciplina

## 🛠️ Tecnologias utilizadas

* Linguagem C

## 📂 Organização

Os conteúdos estão organizados de forma progressiva, acompanhando os tópicos estudados ao longo da disciplina.

## 🚀 Observações

Este repositório tem caráter educacional e foi desenvolvido com o objetivo de aprendizado e prática de programação estruturada.
